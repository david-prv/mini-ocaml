# Mini-OCaml Interpreter
Small interpreter written in Meta-Language OCaml for Object-Language "Mini-OCaml"

## Planned Features
- Lexer / Tokenizer
- Parser
- Check for free Vars
- Type Checking / Guessing
- Evaluation
